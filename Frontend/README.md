# Learning Progress Tracking Dashboard (Frontend)

Frontend-only React + Vite project scaffold for the "Learning Progress Tracking Dashboard".

Run locally:

```bash
# install
npm install

# dev
npm run dev
```

Features:
- React + Vite
- Tailwind CSS with dark mode
- React Router
- Chart.js via `react-chartjs-2`
- Mock data and simulated API (no backend)

Location: The app entry is `src/main.jsx` and pages are in `src/pages`.
